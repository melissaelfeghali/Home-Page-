Flag Matching App

This is an App that displays the flag and name of several countries. The user needs to answer with “True” or “False” to indicate whether the provided name refers to the displayed image. Once the user clicks, he/she will be notified by a Toast whether his choice was right or wrong, then a new question —image along with a name— will be displayed. The app uses SQLite DB to store the names and images. The app displays 5 random questions giving the user the ability to reply with true or false, and displays the score at the end. The user has 5 seconds to answer the displayed question. If he/she fails to answer within 5 seconds, the app will treat it as a wrong answer and move on to display the next question.
